# RAGDrive
Project is created using Electron + Vite. 

## Getting Started

First, clone the repository:

```bash
git clone https://github.com/NidumAI-Inc/ragdrive.git
cd ragdrive
```

Then, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Finally, run the development server:

```bash
npm start
# or
yarn start
# or
pnpm start
```
